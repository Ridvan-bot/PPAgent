# UI/UX Pro Max

Use this skill when the user asks for UI/UX work: design, build, review, or improve interfaces.

- **Product types**: SaaS, e-commerce, portfolio, dashboard, landing page.
- **Styles**: minimal, glassmorphism, brutalism, dark mode, etc.
- **Stacks**: React, Next.js, Vue, Nuxt, Astro, Tailwind, shadcn, etc.

Recommend design systems (colors, typography, patterns), accessibility and UX best practices, and anti-patterns to avoid. For full search/CLI workflow and data, see the [ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) repo.
